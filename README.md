# is-valid-jmbg

Check if [JMBG](http://en.wikipedia.org/wiki/Unique_Master_Citizen_Number) is valid.

## Installation

```sh
npm install is-valid-jmbg --save
```

## Examples

```js
var valid = require('is-valid-jmbg');

valid('1304005111059'); // => true
valid('2607973392147'); // => false
```
